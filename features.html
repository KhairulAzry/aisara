What we are working on

This is only the beginning, and there is much more for us to work on. We think that early feedback will help us make this tool really awesome for you.
To spare you the agony of asking for a long list of ideas for improvement, here are some that we have already thought about, so you can focus on what we have not.

Correction for noise in data

Heatmap plot for visualization of results

Introduce a 2D cross-plotting function to compare objective function and variables

Revision history/changelog for AiCentrica

Introduce SmartBots to aid and facilitate workflow

Advanced options feature to fine tune data input and visualization (bin size, min. and max. values etc)

Surface plot filter on the objective function, allowing focus on range-of-interest

Data transpose

Merging multiple datasheets into a single integrated AiSara project

Ability to handle discrete/categorical in addition to continuous data

Data point labelling on result plot

Ability to search through the solution space

Setup profile, chatbot in the feature

Help documents / videos
